# GyConverse-Client ⚙️

## About

- A web chat application for fun conversation :)
- [server](https://github.com/gywreb/gyconverse-server)

## Built with

- React
- Redux (Thunk, Persist)
- Chakra-UI
- React-Hook-Form
- React Router
- Socket.io (Client)

## Getting Started

#### 1. Install

```bash
# Dependencies Install
npm install
# or
yarn install
```

#### 2. Run

```
yarn start
```

#### 3.Build

```
yarn build
```
